If you use Android Studio create a new device in Virtual Device Manager, you can click the 3 dots at the right of Clone Repository when opening Android Studio. Click Create Virtual Device and choose Pixel 8. After downloading Flutter and Dart on Virtual Studio Code you will have in the bottom-right the writing "No Device", click it and choose "Start Pixel 8". This will start the android emulator, then run the main to start the app.
